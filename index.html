📖 使用帮助

TeamBlog 将帖子存储在 GitHub 仓库的 posts/ 目录，通过 GitHub Pages 免费发布。

1
在 GitHub 创建一个 Public 仓库，开启 GitHub Pages（Settings → Pages → Deploy from branch → main）
2
将本 HTML 文件命名为 index.html 上传到仓库根目录
3
获取 Personal Access Token：GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo 权限
4
在配置栏填入用户名、仓库名、Token，点击「连接」
5
点击「写帖子」，填写内容后「发布到 GitHub」，约 1 分钟后网页自动更新
6
团队成员打开同一网址，填入各自的 Token，即可独立发帖，互不干扰
💡 每篇帖子是独立文件，多人同时发帖不会冲突。Token 仅存在本地浏览器，不会上传。
