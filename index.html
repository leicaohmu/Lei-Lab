🔑 如何获取 GitHub Token

Personal Access Token (PAT) 用于授权网页读写你的 GitHub 仓库。

  1. 登录 GitHub，点击右上角头像 → Settings
  2. 左侧菜单滚动到底部 → Developer settings
  3. 选择 Personal access tokens → Tokens (classic)
  4. 点击 Generate new token (classic)
  5. 填写备注名，Expiration 选择合适期限
  6. 勾选权限：repo（完整仓库访问）
  7. 点击 Generate token，立即复制保存（只显示一次）
⚠️ Token 仅保存在你的浏览器 localStorage 中，不会上传到任何服务器。建议为协作文档单独创建一个仓库。
仓库设置（启用 GitHub Pages）：
  1. 在 GitHub 创建新仓库（Public）
  2. 仓库 Settings → Pages → Source 选择 Deploy from a branch → main / root
  3. 将本 HTML 文件重命名为 index.html 上传到仓库根目录
  4. 几分钟后即可通过 https://用户名.github.io/仓库名 访问
