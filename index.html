⚙ GitHub 设置

📖 使用帮助

TeamBlog 将帖子存储在 GitHub 仓库的 posts/ 目录。

1
创建 Public 仓库，开启 GitHub Pages
2
将本 HTML 文件命名为 index.html 上传到仓库根目录
3
获取 Token：GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo
4
填入配置点击「连接」
5
点击「写帖子」发布，支持导入 .ipynb
💡 Token 仅存本地浏览器，不会上传。
