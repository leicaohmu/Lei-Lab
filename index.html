📖 使用帮助

帖子存储在 GitHub 仓库的 posts/ 目录，通过 GitHub Pages 免费发布。

1
创建 Public 仓库，开启 GitHub Pages（Settings → Pages → Deploy from branch → main）
2
将本文件命名为 index.html 上传到仓库根目录
3
获取 Token：GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo 权限
4
填入用户名、仓库名、Token，点击「🔌 连接」
5
点「✏️ 写帖子」发布内容，约 1 分钟后网页自动更新
💡 每篇帖子是独立文件，多人同时发帖不会冲突。Token 仅存本地浏览器。
