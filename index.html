⚙️ GitHub 设置

📖 使用帮助

TeamBlog 将帖子存储在 GitHub 仓库的 posts/ 目录，通过 GitHub Pages 免费发布。

1
创建 Public 仓库，开启 GitHub Pages（Settings → Pages → Deploy from branch → main）
2
将本 HTML 文件命名为 index.html 上传到仓库根目录
3
获取 Token：GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo 权限
4
填入配置点击「连接」，连接成功后配置栏自动隐藏
5
点击「写帖子」发布，约 1 分钟后 GitHub Pages 自动更新
6
支持导入 .ipynb：写帖子时拖入文件，自动提取标题，完整保留代码输出和图表（Base64内嵌，完全离线无需外部服务）
💡 Token 仅存本地浏览器，不会上传。多人同时发帖不会冲突。
